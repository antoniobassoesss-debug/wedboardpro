# Keyboard Shortcuts Reference

## Selection & Editing

| Shortcut | Action |
|----------|--------|
| `Click` | Select single element |
| `Shift + Click` | Add to selection (multi-select) |
| `Click + Drag` | Box selection |
| `Escape` | Deselect all |

## Element Operations

| Shortcut | Action |
|----------|--------|
| `Delete` / `Backspace` | Delete selected element(s) |
| `Cmd/Ctrl + D` | Duplicate selected element(s) |
| `Cmd/Ctrl + C` | Copy selected to clipboard |
| `Cmd/Ctrl + X` | Cut selected to clipboard |
| `Cmd/Ctrl + V` | Paste from clipboard |
| `Cmd/Ctrl + A` | Select all elements |
| `L` | Toggle lock on selected element |

## Navigation & View

| Shortcut | Action |
|----------|--------|
| `Arrow Keys` | Nudge selected 1px |
| `Shift + Arrow Keys` | Nudge selected 10px |
| `Space (hold)` | Hand tool (pan) |
| `V` | Select tool |
| `H` | Hand tool |
| `S` | Toggle snap |
| `G` | Toggle grid |

## Transformation

| Shortcut | Action |
|----------|--------|
| `R` | Rotate 15° clockwise |
| `Shift + R` | Rotate 15° counter-clockwise |
| `]` | Bring forward |
| `Cmd/Ctrl + ]` | Bring to front |
| `[` | Send backward |
| `Cmd/Ctrl + [` | Send to back |

## Zoom

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + 0` | Fit to content |
| `Cmd/Ctrl + 1` | Zoom to 100% |
| `Cmd/Ctrl + +` | Zoom in |
| `Cmd/Ctrl + -` | Zoom out |

## Undo/Redo

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |

## Notes

- All shortcuts only work when the canvas is focused
- Shortcuts are ignored when typing in input fields
- Multi-select operations apply to all selected elements
- Toast notifications provide feedback for most actions
